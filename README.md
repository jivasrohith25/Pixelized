# Pixelized

Pixelized is a browser-based image transformer that converts uploaded images into stylized ASCII, braille, edge-map, or pixel-art output. It runs entirely on the client using a canvas pipeline, so no server or API keys are required.

## What It Does

- Converts images into multiple render modes: ASCII, braille, edge, and pixel.
- Lets you tune resolution, brightness, contrast, dithering, and glyph size.
- Supports color grading presets (original, matrix, amber, cyber, mono).
- Exports results as TXT, SVG, or PNG.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
